Welcome! This is an open source project that provides Web services to build and perform scientifically interesting computations on Connectome and general graphs.
Services here can either be accessed programmatically or via web forms. Click here to obtain a recent version of the code for the Stand-alone Package that builds Connectome graphs and computes invariants.

Form based use

You may choose to:
  • Build graphs. The input required is a fiber tractography streamline file in MRI Studio DAT format and an optinal NIFTI format altas.
  • Compute Invariants of graphs. Invariants computed include:
    • The Scan Statistic 1
    • The Triangle Count
    • The Clustering Coefficient
    • The Maximum Average Degree
    • The Local Degree Count
    • The Top-k eigenvalues and eigenvectors
  • Convert between graph formats and extract individual invariants from attributed graphs.

Programmatic use

All services can also be accessed programmatically:

  • Upload derivatives to build a graph

    This service's base url is 
    http://mrbrain.cs.jhu.edu/graph-services/upload/
    A single subject graph build example can be found here uploadsubj.py.
    A Multiple subject example is also provided for batch uploading and processing of multiple scans.
    Go to the graph building page for more detailed instructions on programmatic use and example use cases.

  • Compute Invariants of built graphs

    This can be accessed using the base url 

    http://mrbrain.cs.jhu.edu/graph-services/graphupload/
    An example of usage can be found here.
    Go to the compute invariants page for more detailed instructions on programmatic use and example use cases.

  • Convert data file formats

    This can be accessed via the base url 
    http://mrbrain.cs.jhu.edu/graph-services/convert/
    Single and batch conversion examples are available in our examples repository.
    Go to the convert file format page for more detailed instructions on programmatic use and example use cases.
